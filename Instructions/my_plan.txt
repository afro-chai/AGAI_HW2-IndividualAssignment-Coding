News Sentiment Follower vs. Volatility Averse

    News Sentiment Follower
    The crowd knows something. Follow the signal in the noise.
    Recent news sentiment. Overrides price signals when sentiment is strong.
    Alpha Vantage News Sentiment API, plus your market data context
        Note on the News Sentiment Follower: This strategy requires a free Alpha Vantage API key 
        and has limited free requests per day. If you choose this strategy, you must handle the 
        case where the API returns no news for a given ticker. The other four strategies can be 
        completed using only yfinance.

    Volatility Averse
    Survive first, profit second. Avoid turbulence.
    Low-beta stocks, stable price action, low daily swings.
    Standard deviation of daily returns, average true range, max single-day drop in last 90 days
