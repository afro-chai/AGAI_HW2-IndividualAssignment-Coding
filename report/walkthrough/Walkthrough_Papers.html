Part I

Main report — comparative analysis

Source: report/comparative_analysis.tex · Committed PDF: afrocha_HW2_COMP-RPT.pdf

Abstract

This work implements a small multi-agent StockTrader pipeline: two graded strategies—News Sentiment Follower and Volatility Averse—plus a third Moral Trader extension, orchestrated with Microsoft AutoGen and local Ollama inference. Each symbol is analyzed in parallel from the same market snapshot; an evaluator then summarizes agreement and tension across agents. The study uses a compact basket of five U.S. listings chosen for sector spread and defense–geopolitics exposure, including Africa–MENA energy and gold mining angles alongside defense and shipping names. A lightweight news sentiment feed supplies optional headlines; a separate historical exercise scores transparent stand-ins for the two graded philosophies against realized forward returns without replaying the LLMs on past dates.

The full PDF includes strategy rationale, TikZ architecture diagram, stock table, quantitative results, per-ticker discussion, and comparative analysis. Build from LaTeX for the authoritative tables and figures.

Part II

AI use appendix

Source: report/ai_use_appendix.tex · Committed PDF: afrocha_HW2_AI_RPT.pdf

Abstract

This appendix describes how I used generative AI tools—Cursor, ChatGPT, and Ollama—while completing the StockTrader coding assignment. I steered work with written instructions and project context, reviewed every stage of output carefully, and used ChatGPT once with the rubric to stress-test the report for gaps. Ollama was required as the local LLM runtime for the assignment pipeline (strategies and evaluator), separate from my authoring workflow. The LaTeX document includes tools, workflow, representative prompts, sample outputs, file-level attribution, verification notes, and failure-mode discussion.

Tools (summary)

  • Cursor — Primary IDE for code, prompts, docs, and report drafting; all outputs reviewed.
  • ChatGPT — One rubric-aligned gap pass on a late draft; no wholesale ghostwriting.
  • Ollama — Required runtime for strategy and evaluator LLM calls in python -m src.main.
