# walkthrough/

HTML walkthrough for graders: **3 strategy agents + 1 evaluator**, matching the StockTrader pipeline and the TikZ diagram in `comparative_analysis.tex`. All pages link to **`walkthrough.css`** (maroon header, pill nav, lavender CTA, two-column nodes, repository map). Open **`StockTrader_Walkthrough_Home.html`** from a clone of the repo so relative links to `../../src/`, `../../prompts/`, and `../../outputs/` resolve (file:// or a static server). **`Walkthrough_MarketData.html`** is the expanded market-data layer (flow diagram, default tickers table, fifteen substitute-symbol “pop out” tables aligned with **`report/DEFENSE_GEOPOLITICS_UNIVERSE.md`**, plus a link to that worksheet for long-form notes). **`Walkthrough_Artifacts.html`** covers **outputs/** (per-ticker JSON, `summary.json`, optional `backtest.json`), write order, and sample links; the map links to it from the **Artifacts** hub under the diagram. **`Walkthrough_Papers.html`** is the **papers hub**: main report abstract + embedded **`afrocha_HW2_COMP-RPT.pdf`**, a styled break, then the AI use appendix + embedded **`afrocha_HW2_AI_RPT.pdf`**, with links to the `.tex` sources and direct PDF downloads (files live under **`report/`**).
