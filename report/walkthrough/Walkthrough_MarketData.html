Data flow

Fetches and arithmetic only: prices, engineered risk statistics, and optional headline sentiment land in one machine-readable object per run row in DEFAULT_TICKERS.

Vendor fields and computed numbers in; one snapshot out. Strategies never see each other’s outputs—only this shared payload.

Default five tickers (this repository)

Wired for sector diversity, mixed volatility, at least one AFRICOM-relevant name and conflict-adjacent mining — and to cap Alpha Vantage news calls at five per full run.
