\section{Purpose}
This appendix documents how generative AI was used during development and report drafting, with \textbf{file-level attribution} and explicit \textbf{accept / revise / reject} decisions, consistent with course expectations for responsible AI use.

\section{Representative prompts (on-disk)}
The production system loads full prompts from the repository (not pasted inline in code). Key files:
\begin{itemize}
  \item \texttt{prompts/strategy\_a.txt} / \texttt{prompts/strategy\_news\_sentiment.txt} --- News Sentiment Follower
  \item \texttt{prompts/strategy\_b.txt} / \texttt{prompts/strategy\_volatility\_averse.txt} --- Volatility Averse
  \item \texttt{prompts/strategy\_moral\_trader.txt} --- Moral Trader (extension)
  \item \texttt{prompts/evaluator.txt} --- Evaluator
\end{itemize}

\noindent\textbf{Design intent:} strategies must cite numeric fields from the JSON payload; evaluator must explain agreement/disagreement without accessing peer strategy outputs during generation.

\section{Clean run commands (reference)}
From the repository root (after activating the virtual environment):

\begin{verbatim}
pip install -r requirements.txt
$env:PYTHONPATH = (Get-Location).Path
Remove-Item Env:STOCKTRADER_SKIP_LLM -ErrorAction SilentlyContinue
python -m src.main --tickers NVDA,TSLA,JNJ,KO --backtest --ticker-workers 2
\end{verbatim}

\noindent\textbf{Stub mode (CI / no GPU):} set \texttt{STOCKTRADER\_SKIP\_LLM=1} only for smoke tests; not used for final narrative quality.

\noindent\textbf{Optional tuning (Ollama):} see \texttt{.env.example} for \texttt{OLLAMA\_NUM\_PREDICT}, \texttt{OLLAMA\_NUM\_CTX}, \texttt{OLLAMA\_TEMPERATURE}.

\section{What I verified independently}
\begin{itemize}
  \item JSON schema fields match assignment requirements (decision, confidence, justification; evaluator block).
  \item Market data path is non-LLM; strategies do not cross-read before evaluation (code structure).
  \item Empty-news path is exercised when API key is absent (see \texttt{news\_features.note}).
\end{itemize}

\section{One weak or surprising model output (example)}
\textbf{Issue:} occasional language inconsistency when news is missing (e.g., wording that overstates ``high volatility'' when realized vol is comparatively low).

\textbf{Response:} tightened prompts to forbid citing indicators absent from JSON; added evaluator text normalization for contradictory agreement language in code (\texttt{src/evaluator.py}).
