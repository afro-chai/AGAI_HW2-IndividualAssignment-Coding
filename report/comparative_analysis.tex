\begin{abstract}
This report compares two graded behavioral strategies---\textbf{News Sentiment Follower} and \textbf{Volatility Averse}---plus an extension \textbf{Moral Trader} agent, implemented with Microsoft AutoGen and local inference (Ollama). All agents consume the same structured market JSON per ticker. The default equity set is \textbf{five} U.S.-listed names: \textbf{PLTR} (AI / government analytics), \textbf{TTE} and \textbf{GOLD} (\textbf{AFRICOM}-relevant integrated oil and gold mining), \textbf{LMT} (defense prime), and \textbf{FRO} (crude tankers)---see \texttt{report/DEFENSE\_GEOPOLITICS\_UNIVERSE.md}. This keeps \textbf{Alpha Vantage} usage to \textbf{five} \texttt{NEWS\_SENTIMENT} calls per full \texttt{python -m src.main} run (see \texttt{docs/ALPHA\_VANTAGE.md}); the historical backtest path uses \textbf{zero} AV calls. \textbf{Live} decision rows in Table~\ref{tab:results} match the retained 2026-04-12 Ollama outputs for symbols still in the trimmed basket (\texttt{outputs/summary.json}). The \textbf{historical heuristic} scorecard (Table~\ref{tab:backtest}) matches \texttt{outputs/backtest.json} after restricting rows to the same five tickers (115 weekly samples; overall news hit rate 0.3565 vs.\ volatility 0.2783).
\end{abstract}

\section{Strategy Selection and Rationale}

\textbf{Graded pair.} \textbf{News Sentiment Follower} vs.\ \textbf{Volatility Averse}: external narrative and sentiment (when news exists) versus measurable path risk (volatility, drawdowns, ATR).

\textbf{Defense--geopolitics framing.} The ticker universe is documented in \texttt{report/DEFENSE\_GEOPOLITICS\_UNIVERSE.md}. The default run uses a \textbf{small, diverse} basket with explicit \textbf{AFRICOM} hooks on \textbf{TTE} (Africa--MENA oil and LNG) and \textbf{GOLD} (Mali / DRC gold exposure narratives), plus U.S.\ defense software, a prime, and tanker beta.

\textbf{Extension.} \textbf{Moral Trader} adds a third parallel branch (narrative opacity / accountability), evaluated only after A and B finish for the same ticker.

\section{System Architecture}

\subsection{Orchestration logic}
\begin{itemize}
  \item \textbf{Market data (no LLM):} \texttt{src/market\_data.py} pulls yfinance history and derived features; optionally calls Alpha Vantage \texttt{NEWS\_SENTIMENT} once per ticker. API key: \texttt{ALPHAVANTAGE\_API\_KEY} in repo-root \texttt{.env} and/or gitignored \texttt{secrets/alphavantage\_api\_key.txt} (see \texttt{secrets/README.md}).
  \item \textbf{Strategy agents (LLM):} three \texttt{AssistantAgent} instances run in parallel per ticker with structured outputs (decision, confidence, justification).
  \item \textbf{Isolation:} strategies receive the same JSON context and do not see each other's outputs before the evaluator.
  \item \textbf{Evaluator (LLM):} compares outputs and writes consensus vs.\ divergence analysis.
  \item \textbf{Persistence:} per-ticker JSON under \texttt{outputs/}, plus \texttt{summary.json} and optional \texttt{backtest.json}.
\end{itemize}

\subsection{Evaluator}

For all five retained tickers, the evaluator records \textbf{unanimous HOLD} with \texttt{agents\_agree: true}. On names where Alpha Vantage returned empty feeds, Moral Trader may still surface \texttt{news\_features.note} (rate limits / API hygiene) in its accountability framing.

\section{Patterns of Agreement and Disagreement}

\textbf{Reading \texttt{summary.json}.} For the five-ticker summary, \texttt{total\_agreements} $= 5$ and \texttt{total\_disagreements} $= 0$: unanimous HOLD across A/B/C on each symbol in the trimmed basket.

\textbf{Confidence spread.} Largest spread is on \textbf{GOLD} (3 points: Moral at 5 vs.\ both graded agents at 8). Other names show Moral at 6 vs.\ 8/8 (spread 2).

\textbf{Alpha Vantage budget.} Five tickers imply five \texttt{NEWS\_SENTIMENT} calls per full run---well inside a typical free daily cap if you avoid extra tooling bursts; optional \texttt{ALPHAVANTAGE\_MIN\_INTERVAL\_SEC} and \texttt{--ticker-workers 1} remain available if you still see throttling.

\section{Failure or Surprise Case}

\textbf{Operational note:} when the free tier returns empty feeds, Strategy~A rarely has non-null \texttt{avg\_sentiment\_score}; \texttt{src/market\_data.py} scrubs echoed key material from informational strings before JSON is written.

\textbf{Behavioral surprise:} \textbf{FRO} still produced unanimous HOLD with ``low volatility'' language in Strategy~B---consistent with bounded realized vol on the window rather than Hormuz headlines in the numeric slice alone.

\section{Reflection}

For a one-month, \$10{,}000 thought experiment with this basket, I would still weight \textbf{Volatility Averse} heavily when news is empty: Strategy~A's differentiator collapses toward shared OHLC-derived heuristics, whereas Strategy~B is explicitly built around realized turbulence and drawdown statistics. With a data tier that restores reliable \texttt{NEWS\_SENTIMENT} coverage, I would revisit Strategy~A on the same five symbols.
