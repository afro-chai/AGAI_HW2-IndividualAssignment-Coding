\begin{abstract}
This report compares two graded behavioral strategies---\textbf{News Sentiment Follower} and \textbf{Volatility Averse}---plus an extension \textbf{Moral Trader} agent, implemented with Microsoft AutoGen and local inference (Ollama). All agents consume the same structured market JSON per ticker. The analysis reflects the latest committed run in \Repo{} for NVDA, TSLA, JNJ, and KO (\textbf{run date 2026-04-11}), produced on a laptop with live Ollama (\texttt{STOCKTRADER\_SKIP\_LLM} unset; wall clock on the order of twenty to twenty-five minutes per \texttt{logs/full\_run\_latest.txt}). Quantitative summaries come from \texttt{outputs/summary.json} and the heuristic bonus artifact \texttt{outputs/backtest.json}.
\end{abstract}

\section{Strategy Selection and Rationale}

\textbf{Graded pair.} I selected \textbf{News Sentiment Follower} and \textbf{Volatility Averse} because they embody a clean philosophical contrast: the first prioritizes \emph{external narrative and sentiment} when available, while the second prioritizes \emph{internal stability and tail risk} from price paths (volatility, drawdowns, ATR).

\textbf{Why this pairing.} I expected disagreement when headline sentiment is strong but price action is choppy (news-driven optimism vs.\ risk-off posture), and agreement when both signals are ambiguous (e.g., quiet tape, middling volatility).

\textbf{Extension.} I added \textbf{Moral Trader} as a third parallel branch (bonus/capstone-oriented lens on narrative opacity and accountability), evaluated alongside the core pair.

\section{System Architecture}

\subsection{Orchestration logic}
\begin{itemize}
  \item \textbf{Market data (no LLM):} \texttt{src/market\_data.py} pulls yfinance history and derived features; optionally calls Alpha Vantage for news sentiment.
  \item \textbf{Strategy agents (LLM):} three \texttt{AssistantAgent} instances run in parallel per ticker with structured outputs (decision, confidence, justification).
  \item \textbf{Isolation:} strategies receive the same JSON context and do not see each other's outputs before the evaluator.
  \item \textbf{Evaluator (LLM):} compares outputs and writes consensus vs.\ divergence analysis.
  \item \textbf{Persistence:} per-ticker JSON under \texttt{outputs/}, plus \texttt{summary.json} and optional \texttt{backtest.json}.
\end{itemize}

\section{Patterns of Agreement and Disagreement}

\textbf{Observed pattern in this run: broad agreement.} For all four tickers, strategies A/B/C agreed on \textbf{HOLD}, yielding \texttt{total\_agreements = 4} and \texttt{total\_disagreements = 0} in \texttt{outputs/summary.json}.

\textbf{Interpretation (not just restatement).} Convergence is plausible given:
\begin{itemize}
  \item \textbf{Missing news channel on this run:} every ticker file still shows \texttt{articles\_count = 0} and \texttt{note: "no API key"}, so the News agent cannot exploit sentiment overrides and defaults to HOLD language grounded in price and volatility fields. The committed laptop log (\texttt{logs/full\_run\_latest.txt}) explicitly warns that if this persists, the Python process may not have loaded repo-root \texttt{.env} (or the key was absent on that machine).
  \item \textbf{HOLD despite heterogeneous tape:} TSLA's 30-day return is $-14.44\%$ while NVDA is slightly up and JNJ/KO are comparatively quiet; all three strategies still output HOLD---the models emphasize bounded realized volatility and the absence of a sentiment lever rather than aggressively shorting weakness.
\end{itemize}

\textbf{Confidence spread (quantitative nuance).} Moral Trader's confidence is lowest on JNJ (3) while A/B stay at 8, producing the widest three-way spread (5) in the table; KO shows perfect alignment (all 8).

\textbf{Where sharper disagreement would likely appear.} With a working Alpha Vantage key in the runtime environment (and respecting rate limits), non-null \texttt{avg\_sentiment\_score} and article text could push Strategy A toward BUY/SELL while Strategy B remains risk-focused---especially on high-beta names.

\section{Failure or Surprise Case}

\textbf{Surprise:} after a full live Ollama run, the portfolio is still uniformly HOLD even where the tape is not uniform---notably TSLA's large negative 30-day return coexists with Strategy~B citing ``low turbulence'' and all agents choosing HOLD.

\textbf{Likely causes:} (1) \texttt{news\_features} remained empty (\texttt{no API key} in JSON), so Strategy~A lacks its primary differentiated signal; (2) local model conservatism on structured JSON; (3) Strategy~B's thresholds mapping moderate realized vol to HOLD; (4) evaluator normalization when decisions are unanimous.

\textbf{Mitigation for future runs:} ensure \texttt{.env} is loaded on the machine that runs \texttt{python -m src.main} so \texttt{ALPHAVANTAGE\_API\_KEY} is visible; consider tickers or windows with stronger divergence between sentiment and vol; prompt tuning if HOLD is over-selected.

\section{Reflection}

If I had to follow \textbf{one graded strategy for one month with \$10{,}000}, I would choose \textbf{Volatility Averse} on this implementation because, without reliable headline sentiment, the News agent is partially blind; the volatility-focused agent at least anchors on measurable path risk.

A \textbf{hybrid} would gate sentiment: only allow Strategy A to override when $\lvert\texttt{avg\_sentiment\_score}\rvert$ exceeds a threshold \emph{and} news volume is non-trivial; otherwise defer to Strategy B's risk checks.
