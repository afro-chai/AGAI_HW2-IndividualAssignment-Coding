\begin{abstract}
This report compares two graded behavioral strategies---\textbf{News Sentiment Follower} and \textbf{Volatility Averse}---plus an extension \textbf{Moral Trader} agent, implemented with Microsoft AutoGen and local inference (Ollama). All agents consume the same structured market JSON per ticker. The default equity set is \textbf{ten} U.S.-listed names spanning defense primes, AI analytics, semiconductors, \textbf{AFRICOM-relevant} energy and gold (TotalEnergies, Eni, Barrick), cybersecurity, and crude tankers (see \texttt{report/DEFENSE\_GEOPOLITICS\_UNIVERSE.md}). The \textbf{final committed run} is dated \textbf{2026-04-12}: wall clock about \textbf{25 minutes} on the reference laptop (\texttt{--ticker-workers 2}, live Ollama, \texttt{STOCKTRADER\_SKIP\_LLM} unset). Alpha Vantage returned \textbf{empty news feeds} for most symbols after the free tier's \textbf{daily request budget} was exhausted mid-batch; \texttt{news\_features.note} carries Alpha Vantage's own explanation (API keys are \textbf{scrubbed} in \texttt{src/market\_data.py} before JSON is written). Quantitative rows below match \texttt{outputs/summary.json} for that run: \textbf{ten unanimous HOLD} decisions (\texttt{total\_agreements}=10, \texttt{total\_disagreements}=0).
\end{abstract}

\section{Strategy Selection and Rationale}

\textbf{Graded pair.} \textbf{News Sentiment Follower} vs.\ \textbf{Volatility Averse}: external narrative and sentiment (when news exists) versus measurable path risk (volatility, drawdowns, ATR).

\textbf{Defense--geopolitics framing.} The ticker universe is documented in \texttt{report/DEFENSE\_GEOPOLITICS\_UNIVERSE.md}. The default run uses symbols expected to carry \textbf{defense budget}, \textbf{regional conflict}, and \textbf{energy security} headlines---including \textbf{three AFRICOM-heavy picks} (\textbf{TTE}, \textbf{E}, \textbf{GOLD}) for Africa/MENA oil and conflict-adjacent mining narratives.

\textbf{Extension.} \textbf{Moral Trader} adds a third parallel branch (narrative opacity / accountability), evaluated only after A and B finish for the same ticker.

\section{System Architecture}

\subsection{Orchestration logic}
\begin{itemize}
  \item \textbf{Market data (no LLM):} \texttt{src/market\_data.py} pulls yfinance history and derived features; optionally calls Alpha Vantage \texttt{NEWS\_SENTIMENT}. API key: \texttt{ALPHAVANTAGE\_API\_KEY} in repo-root \texttt{.env} and/or gitignored \texttt{secrets/alphavantage\_api\_key.txt} (see \texttt{secrets/README.md}).
  \item \textbf{Strategy agents (LLM):} three \texttt{AssistantAgent} instances run in parallel per ticker with structured outputs (decision, confidence, justification).
  \item \textbf{Isolation:} strategies receive the same JSON context and do not see each other's outputs before the evaluator.
  \item \textbf{Evaluator (LLM):} compares outputs and writes consensus vs.\ divergence analysis.
  \item \textbf{Persistence:} per-ticker JSON under \texttt{outputs/}, plus \texttt{summary.json} and optional \texttt{backtest.json}.
\end{itemize}

\subsection{Evaluator}

For all ten tickers on 2026-04-12, the evaluator records \textbf{unanimous HOLD} with \texttt{agents\_agree: true}. \texttt{pattern\_note} appears on several names (e.g.\ PLTR) where Moral Trader flags \textbf{data-access / rate-limit language} in \texttt{news\_features.note} as an accountability theme even though A/B stayed anchored on price and volatility metrics.

\section{Patterns of Agreement and Disagreement}

\textbf{Reading \texttt{summary.json}.} For 2026-04-12, \texttt{total\_agreements} $= 10$ and \texttt{total\_disagreements} $= 0$: every ticker shows unanimous HOLD across A/B/C despite mixed tape (e.g.\ PLTR down $\approx 15.5\%$ over 30 days vs.\ quieter defense primes). This basket was chosen to \emph{invite} disagreement, but the local models converged on conservative action once headline sentiment was largely absent.

\textbf{Confidence spread.} Largest spread is on \textbf{GOLD} (3 points: Moral at 5 vs.\ both graded agents at 8). Several names show Moral at 6 vs.\ 8/8 (spread 2); \textbf{E} is tightest (Moral 7 vs.\ 8/8).

\textbf{Alpha Vantage at scale.} Ten tickers imply ten \texttt{NEWS\_SENTIMENT} calls per pass; the committed run hit the \textbf{free daily cap} partway through, so most files show \texttt{articles\_count = 0} with an informational \texttt{note}. Mitigations: premium tier, split runs across days, \texttt{ALPHAVANTAGE\_MIN\_INTERVAL\_SEC}, and/or \texttt{--ticker-workers 1} to reduce burst pressure.

\section{Failure or Surprise Case}

\textbf{Operational surprise:} Alpha Vantage's response body included a \textbf{daily limit} explanation for most tickers after the 25-calls/day budget was exhausted, so Strategy~A rarely had non-null \texttt{avg\_sentiment\_score} in the retained JSON. (The upstream message can echo a raw key string; this repository scrubs that pattern in \texttt{src/market\_data.py} and redacts committed outputs---\textbf{rotate the key} if it was ever exposed outside the scrubbed copy.)

\textbf{Behavioral surprise:} even \textbf{FRO} (tankers, Hormuz-beta framing in the universe note) produced unanimous HOLD with ``low volatility'' language in Strategy~B---consistent with bounded realized vol on the window rather than geopolitical drama in the numeric slice alone.

\textbf{Evaluator surprise:} Moral Trader sometimes elevates \texttt{news\_features.note} (rate limits / API hygiene) into the moral frame when articles are empty, which is honest as a \emph{governance} story but not the same as detecting fraud in prices.

\section{Reflection}

For a one-month, \$10{,}000 thought experiment with this basket, I would still choose \textbf{Volatility Averse} first: when Alpha Vantage is empty or quota-blocked, Strategy~A's differentiator collapses toward the same OHLC-derived heuristics, whereas Strategy~B is explicitly built around realized turbulence and drawdown statistics. If I upgraded to a data tier that restored reliable \texttt{NEWS\_SENTIMENT} coverage, I would revisit Strategy~A on the same symbols---the basket was chosen partly to stress-test headline disagreement once feeds populate.
